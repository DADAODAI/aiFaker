# Logo 图标文件

请在此目录放置各 AI 品牌的 logo 图片。

## 命名约定

| 文件名         | 对应预设        |
|---------------|----------------|
| chatgpt.png   | ChatGPT 深/浅色 |
| deepseek.png  | DeepSeek       |
| claude.png    | Claude         |
| gemini.png    | Gemini         |

## 要求

- 格式：PNG（透明背景最佳）
- 建议尺寸：88×88 px 或更大的正方形
- 渲染时会被裁剪为圆形并缩放到 44×44 px

如果没有放对应的 logo 文件，渲染器会退回到带颜色的圆形 + 文字头像。
